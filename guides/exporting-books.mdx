Once you've created a story, StoryFox offers multiple export formats so you can share, print, or preserve your book.

## Export formats

StoryFox supports three export options:

  <Card title="EPUB" icon="book.closed">

## Exporting as PDF

PDF is the most versatile format — use it for printing, emailing, or viewing on any device.

<Steps>
  <Step title="Open your book">
    In the reader view, navigate to the book you want to export. Make sure all illustrations have finished generating.
  </Step>

  <Step title="Click Export to PDF">
    In the toolbar, click the **Export to PDF** button (document icon with arrow).
  </Step>

  <Step title="Choose save location">
    A save dialog appears. Choose where to save your PDF:
    - Desktop (for quick access)
    - Documents folder (for organization)
    - Cloud storage like iCloud Drive (for backup)
    
    The filename defaults to your book's title with `.pdf` extension.
  </Step>

  <Step title="Save the file">
    Click **Save**. StoryFox renders your book at 300 DPI print resolution.
  </Step>
</Steps>

### PDF features

StoryFox PDFs include:

- **Print-ready resolution**: 300 DPI ensures crisp, professional quality
- **Full book layout**:
  - Cover page with title and illustration
  - Story pages with text and images
  - "The End" page with moral/theme
- **Embedded fonts**: Text renders consistently on all devices
- **Correct dimensions**: Matches your selected book format (Standard Square, Landscape, Portrait, or Small Square)

### Print dimensions

| Format | Size | Resolution |
|--------|------|------------|
| Standard Square | 8.5" × 8.5" | 2550 × 2550 px |
| Landscape | 11" × 8.5" | 3300 × 2550 px |
| Portrait | 8.5" × 11" | 2550 × 3300 px |
| Small Square | 6" × 6" | 1800 × 1800 px |

<Tip>
  **Printing tip**: Use the Standard Square or Landscape format for the best printing experience. Small Square is designed for compact board books.
</Tip>

## Exporting as EPUB

EPUB is the standard ebook format supported by most e-readers and book apps.

<Steps>
  <Step title="Open your book">
    Navigate to the completed book in reader view.
  </Step>

  <Step title="Click Export to EPUB">
    In the toolbar, click the **Export to EPUB** button.
  </Step>

  <Step title="Choose save location">
    Select where to save the EPUB file. The filename defaults to `[Title].epub`.
  </Step>

  <Step title="Save and open">
    After saving, you can:
    - Double-click the EPUB to open in Apple Books
    - Transfer to your e-reader device
    - Share via email or cloud storage
  </Step>
</Steps>

### EPUB features

StoryFox EPUBs include:

- **Reflowable text**: Text adapts to screen size and reader settings
- **Embedded images**: All illustrations are included at web-optimized resolution
- **Metadata**: Title, author, and description for library organization
- **Chapter structure**: Each page becomes a chapter for easy navigation
- **EPUB 3 standard**: Compatible with modern e-readers

### Where to read EPUB files

<CardGroup cols={2}>
  <Card title="Apple Books" icon="book">
    macOS and iOS — opens automatically when you double-click the EPUB
  </Card>
  <Card title="Kindle" icon="kindle">
    Amazon devices — use Kindle Previewer or email to your Kindle
  </Card>
  <Card title="Kobo" icon="tablet">
    Kobo e-readers — transfer via USB or Kobo Desktop app
  </Card>
  <Card title="Calibre" icon="books">
    Cross-platform ebook manager — import and convert to other formats
  </Card>
</CardGroup>

<Warning>
  **EPUB limitations**: Some e-readers may resize images or reflow text in unexpected ways. For pixel-perfect layout, use PDF instead.
</Warning>

## Exporting individual pages as images

Export specific pages as PNG images for sharing on social media, printing as posters, or using in presentations.

<Steps>
  <Step title="Navigate to the page">
    Use the arrow buttons or swipe to go to the page you want to export.
  </Step>

  <Step title="Click Export Page as Image">
    In the toolbar, click the **Export Page** button (image icon with arrow).
  </Step>

  <Step title="Choose save location">
    The filename defaults to:
    - `[Title] - Cover.png` (for title page)
    - `[Title] - Page [number].png` (for story pages)
    - `[Title] - The End.png` (for final page)
  </Step>

  <Step title="Save the PNG">
    Click **Save**. StoryFox renders the full page layout as a high-resolution PNG.
  </Step>
</Steps>

### Image export features

- **High resolution**: Matches your book format's print dimensions (see table above)
- **Full page layout**: Includes text, illustration, and any decorative elements
- **PNG format**: Lossless compression preserves quality
- **Perfect for sharing**: Post to Instagram, print as wall art, or use in presentations

<Tip>
  **Social media tip**: Landscape format (11" × 8.5") exports work great for Instagram posts. Portrait format fits well in Instagram Stories.
</Tip>

## Managing exported files

### Organizing your exports

Create a folder structure to keep exports organized:

```
~/Documents/StoryFox Books/
├── Luna's Forest Adventure/
│   ├── Luna's Forest Adventure.pdf
│   ├── Luna's Forest Adventure.epub
│   └── Pages/
│       ├── Cover.png
│       ├── Page 1.png
│       └── Page 2.png
└── The Brave Little Turtle/
    ├── The Brave Little Turtle.pdf
    └── ...
```

### Cloud backup

Back up your exports to cloud storage:

- **iCloud Drive**: Automatic sync across all Apple devices
- **Dropbox**: Cross-platform access and sharing links
- **Google Drive**: Generous free storage and easy sharing

<Info>
  StoryFox books are saved in the app's library, but exports are separate files. Always export important books to your Documents folder or cloud storage for safekeeping.
</Info>

## Printing your book

### Home printing

For best results when printing at home:

1. Export as PDF using Standard Square or Landscape format
2. Print on high-quality paper (32 lb / 120 gsm or heavier)
3. Use Color printing mode (not grayscale)
4. Select "Actual Size" in print settings (not "Fit to Page")

### Professional printing

StoryFox PDFs are print-ready at 300 DPI. Services like:

- **Blurb**: Hardcover and softcover photo books
- **Lulu**: Print-on-demand with ISBN options
- **Local print shops**: One-off prints or small batches

<Warning>
  **Copyright notice**: Books created with StoryFox are for personal use. If you plan to sell or distribute widely, ensure you have appropriate rights to the AI-generated content.
</Warning>

## Troubleshooting export issues

<AccordionGroup>
  <Accordion title="Export button is grayed out">
    This happens when:
    - Illustrations are still generating (wait for completion)
    - No book is loaded in the reader
    - The book failed to generate properly
    
    **Fix**: Make sure generation completed successfully before exporting.
  </Accordion>

  <Accordion title="PDF looks blurry when printed">
    Possible causes:
    - Printer is set to draft/economy mode
    - Paper quality is too low
    - Print settings scaled the PDF incorrectly
    
    **Fix**: Use high-quality mode, heavier paper, and "Actual Size" print setting.
  </Accordion>

  <Accordion title="EPUB images don't display">
    Some e-readers struggle with embedded images:
    - Older Kindle devices have limited image support
    - File size may be too large for the device
    
    **Fix**: Try opening in Apple Books or Calibre first. Consider using PDF for image-heavy books.
  </Accordion>

  <Accordion title="Can't save to chosen location">
    Permission issues may prevent saving:
    - Folder is read-only
    - Disk is full
    - macOS sandboxing restrictions
    
    **Fix**: Choose a different location like Desktop or Documents folder.
  </Accordion>

  <Accordion title="Exported file is huge">
    File size depends on:
    - Page count (more pages = larger file)
    - Image complexity (detailed illustrations increase size)
    - Format (Standard Square is larger than Small Square)
    
    Typical sizes:
    - PDF: 5-20 MB for 4-8 pages
    - EPUB: 3-15 MB
    - PNG: 2-4 MB per page
    
    **Fix**: This is normal for high-resolution images. Compress PDFs using Preview > Export > Reduce File Size if needed.
  </Accordion>
</AccordionGroup>

## Export technical details

For developers and advanced users:

### PDF rendering

- Renderer: `StoryPDFRenderer` (macOS CoreGraphics)
- Resolution: 300 DPI for print quality
- Color space: sRGB
- Implementation: `/workspace/source/Shared/Generation/PDFRendering.swift`
- Platform-specific: `/workspace/source/macOS/Views/MacExportView.swift:8`

### EPUB rendering

- Renderer: `StoryEPUBRenderer`
- Format: EPUB 3
- Structure: ZIP archive with XHTML pages
- Implementation: `/workspace/source/Shared/Generation/EPUBRendering.swift`
- ZIP building: `ZIPBuilder` utility

### Image export

- Renderer: `PageImageRenderer`
- Format: PNG (lossless)
- Color depth: 24-bit RGB
- Implementation: Uses CoreGraphics `CGImageDestination`
- Code: `/workspace/source/macOS/Views/MacExportView.swift:72`
