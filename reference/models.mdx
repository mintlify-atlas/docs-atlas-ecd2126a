StoryFox uses Swift structs and enums to represent storybooks, their pages, formats, and illustration styles. All models are defined in `Shared/Models/`.

## StoryBook

The primary data model representing a complete storybook with its pages.

<ParamField path="title" type="String" required>
  A captivating title for the children's storybook
</ParamField>

<ParamField path="authorLine" type="String" required>
  The author attribution line (defaults to "Written by StoryFox")
</ParamField>

<ParamField path="moral" type="String" required>
  A one-sentence summary of the story's moral or theme
</ParamField>

<ParamField path="characterDescriptions" type="String" required>
  Brief visual reference for each main character. One line per character: name - specific species or breed, colors, clothing, one distinguishing feature.
</ParamField>

<ParamField path="pages" type="[StoryPage]" required>
  Array of story pages with text and image prompts
</ParamField>

### Example

```swift StoryBook.swift:5-20
@Generable
struct StoryBook {
    @Guide(description: "A captivating title for this children's storybook")
    var title: String

    @Guide(description: "The author attribution line. Always use 'Written by StoryFox' unless the user explicitly provides an author name.")
    var authorLine: String

    @Guide(description: "A one-sentence summary of the story's moral or theme")
    var moral: String

    @Guide(description: "A brief visual reference for each main character...")
    var characterDescriptions: String

    @Guide(description: "The story pages array — you MUST generate the exact number of pages requested.")
    var pages: [StoryPage]
}
```

## StoryPage

Represents a single page in a storybook with narrative text and illustration prompt.

<ParamField path="pageNumber" type="Int" required>
  The page number, starting from 1
</ParamField>

<ParamField path="text" type="String" required>
  2-4 sentences of story text for this page. Uses simple vocabulary appropriate for children ages 3-8.
</ParamField>

<ParamField path="imagePrompt" type="String" required>
  Detailed visual scene description for generating the page illustration. Must include character species/breed and key visual traits in every prompt for consistency.
</ParamField>

### Example

```swift StoryBook.swift:22-32
@Generable
struct StoryPage {
    @Guide(description: "The page number, starting from 1")
    var pageNumber: Int

    @Guide(description: "2-4 sentences of story text for this page...")
    var text: String

    @Guide(description: "A detailed visual scene description for generating an illustration...")
    var imagePrompt: String
}
```

</ParamField>

<ParamField path="title" type="String" required>
  Story title
</ParamField>

<ParamField path="authorLine" type="String" required>
  Author attribution
</ParamField>

<ParamField path="moral" type="String" required>
  Story moral/theme
</ParamField>

<ParamField path="createdAt" type="Date" required>
  Creation timestamp
</ParamField>

<ParamField path="formatRawValue" type="String" required>
  Raw value of BookFormat enum
</ParamField>

<ParamField path="styleRawValue" type="String" required>
  Raw value of IllustrationStyle enum
</ParamField>

<ParamField path="coverImageData" type="Data?">

</ParamField>

<ParamField path="isFavorite" type="Bool" default="false">

<ParamField path="displayOrder" type="Int" default="0">

<ParamField path="characterDescriptions" type="String" default='""'>

<ParamField path="imageModelName" type="String" default='""'>

      let storyPages = pages
          .sorted { $0.pageNumber < $1.pageNumber }
          .map { page in

<Expandable title="toImages()">
  Reconstructs the `[Int: CGImage]` dictionary from stored PNG data.

  ```swift StoredStorybook.swift:160-175
  func toImages() -> [Int: CGImage] {
      var result: [Int: CGImage] = [:]
      if let coverData = coverImageData, let cgImage = cgImageFromData(coverData) {
          result[0] = cgImage
      }
      for page in pages {
          if let data = page.imageData, let cgImage = cgImageFromData(data) {
              result[page.pageNumber] = cgImage
          }
      }
      return result
  }
  ```
</Expandable>

## StoredPage

SwiftData model representing a single persisted page with text, prompt, and image data.

<ParamField path="pageNumber" type="Int" required>
  Page number within the story
</ParamField>

<ParamField path="text" type="String" required>
  Story text for this page
</ParamField>

<ParamField path="imagePrompt" type="String" required>
  Illustration prompt used to generate the page image
</ParamField>

<ParamField path="imageData" type="Data?">
  PNG data for the generated illustration
</ParamField>

### Example

```swift StoredStorybook.swift:78-91
@Model
final class StoredPage {
    var pageNumber: Int
    var text: String
    var imagePrompt: String
    var imageData: Data?

    init(pageNumber: Int, text: String, imagePrompt: String, imageData: Data? = nil) {
        self.pageNumber = pageNumber
        self.text = text
        self.imagePrompt = imagePrompt
        self.imageData = imageData
    }
}
```
