StoryFox exports your completed storybooks to professional PDF and EPUB formats. Both formats include print-ready dimensions, custom typography, and the warm "Library at Dusk" color theme.

## Export formats

<Tabs>
  <Tab title="PDF">
    **Print-ready PDF** for professional printing or digital sharing
    
    - 300 DPI print resolution
    - Embedded Georgia fonts
    - Rounded image corners
    - Title page, story pages, and end page
    - Page numbers and decorative ornaments
  </Tab>
  
  <Tab title="EPUB">
    **EPUB 3.0 Fixed Layout** for digital reading on e-readers
    
    - Compatible with Apple Books, Kindle, and other readers
    - Fixed-layout pages (not reflowable)
    - Embedded images and fonts
    - Navigation document
    - Cover image with text overlay
  </Tab>
</Tabs>

## PDF export architecture

PDF rendering uses Core Graphics for precise layout control:

```swift
struct StoryPDFRenderer: PDFRendering {
    func render(storybook: StoryBook, images: [Int: CGImage], format: BookFormat) -> Data {
        let pageSize = format.dimensions
        let pdfData = NSMutableData()
        var mediaBox = CGRect(origin: .zero, size: pageSize)
        
        let consumer = CGDataConsumer(data: pdfData)
        let context = CGContext(consumer: consumer, mediaBox: &mediaBox, nil)
        
        renderTitlePage(context: context, storybook: storybook, ...)
        
        for page in storybook.pages {
            renderContentPage(context: context, page: page, ...)
        }
        
        renderEndPage(context: context, storybook: storybook, ...)
        
        context.closePDF()
        return pdfData as Data
    }
}
```

## Image fitting

Images are aspect-fit to preserve their proportions:

```swift
private func fitImage(_ image: CGImage, in rect: CGRect) -> CGRect {
    let imageAspect = CGFloat(image.width) / CGFloat(image.height)
    let rectAspect = rect.width / rect.height
    
    if imageAspect > rectAspect {
        // Image is wider - fit to width
        let height = rect.width / imageAspect
        let y = rect.midY - height / 2
        return CGRect(x: rect.minX, y: y, width: rect.width, height: height)
    } else {
        // Image is taller - fit to height
        let width = rect.height * imageAspect
        let x = rect.midX - width / 2
        return CGRect(x: x, y: rect.minY, width: width, height: rect.height)
    }
}
```

## EPUB export architecture

EPUB export builds a ZIP archive with EPUB 3.0 structure:

```swift
struct StoryEPUBRenderer: EPUBRendering {
    func render(storybook: StoryBook, images: [Int: CGImage], format: BookFormat) -> Data {
        var zip = ZIPBuilder()
        
        // 1. mimetype - must be first, uncompressed
        zip.addStored(path: "mimetype", data: Data("application/epub+zip".utf8))
        
        // 2. META-INF/container.xml
        zip.addDeflated(path: "META-INF/container.xml", data: containerXML)
        
        // 3. CSS
        zip.addDeflated(path: "OEBPS/styles.css", data: stylesheet)
        
        // 4. Images (JPEG at 0.85 quality)
        for page in storybook.pages {
            if let img = images[page.pageNumber], let jpeg = cgImageToJPEGData(img) {
                zip.addStored(path: "OEBPS/images/page-\(page.pageNumber).jpg", data: jpeg)
            }
        }
        
        // 5. XHTML pages
        // 6. Navigation document
        // 7. OPF package document
        
        return zip.finalize()
    }
}
```

## Cover image compositing

EPUB covers include text overlay so the title appears in bookshelf views:

```swift
private func compositeCoverImage(
    _ image: CGImage,
    title: String,
    author: String
) -> CGImage {
    // 1. Draw original illustration
    ctx.draw(image, in: fullRect)
    
    // 2. Draw gradient scrim over bottom 40%
    let gradient = CGGradient(
        colorsSpace: colorSpace,
        colors: [
            CGColor(red: 0.08, green: 0.05, blue: 0.03, alpha: 0.85),
            CGColor(red: 0.08, green: 0.05, blue: 0.03, alpha: 0.0)
        ] as CFArray,
        locations: [0.0, 1.0]
    )
    ctx.drawLinearGradient(gradient, ...)
    
    // 3. Draw title text in white
    // 4. Draw author text in semi-transparent white
    
    return ctx.makeImage() ?? image
}
```

<Note>
  The text overlay ensures the book title is visible when browsing your library in e-reader apps like Apple Books.
</Note>

## Image compression

Both formats use optimized image compression:

<Tabs>
  <Tab title="PDF">
    Images are embedded as-is (already compressed by the image generator)
  </Tab>
  
  <Tab title="EPUB">
    Images are converted to JPEG at 85% quality:
    
    ```swift
    private func cgImageToJPEGData(
        _ image: CGImage,
        quality: CGFloat = 0.85
    ) -> Data? {
        let mutableData = NSMutableData()
        let destination = CGImageDestinationCreateWithData(
            mutableData as CFMutableData,
            "public.jpeg" as CFString,
            1,
            nil
        )
        let options: [CFString: Any] = [
            kCGImageDestinationLossyCompressionQuality: quality
        ]
        CGImageDestinationAddImage(destination, image, options as CFDictionary)
        CGImageDestinationFinalize(destination)
        return mutableData as Data
    }
    ```
  </Tab>
</Tabs>
