StoryFox transforms simple story concepts into complete children's books using Apple's on-device language models. The generation system creates narrative text, character descriptions, and page-by-page content with built-in safety and retry mechanisms.

## How it works

Story generation uses Apple's Foundation Models to create structured storybook content directly on your Mac. The system streams results in real-time so you can watch your story come to life.

<Steps>
  <Step title="Enter your story concept">
    Describe the story you want to create in a few sentences. For example: "A curious fox named Luna discovers a magical library in the forest."
  </Step>
  
  <Step title="Choose page count">
    Select how many story pages you want (typically 4-12 pages for children's books).
  </Step>
  
  <Step title="AI generates your story">
    The on-device language model creates:
    - A captivating title
    - Page-by-page narrative text
    - Character descriptions
    - A story moral or theme
  </Step>
</Steps>

## Real-time streaming

StoryFox streams partial results as the AI generates your story, providing instant feedback:

```swift
let textStream = textSession.streamResponse(
    to: textPrompt,
    generating: TextOnlyStoryBook.self,
    options: textOptions
)

for try await snapshot in textStream {
    let partial = snapshot.content
    let pages = partial.pages ?? []
    let previewText = pages.compactMap(\.text).joined(separator: "\n\n")
    onProgress(previewText)
}
```

## Automatic retry on safety filters

The generation system includes built-in retry logic for false-positive content safety blocks:

<Note>
  If the system detects a guardrail violation (often a false positive for innocent children's content), it automatically retries the generation with a fresh session up to 3 times.
</Note>

```swift
for attempt in 0...GenerationConfig.guardrailRetryAttempts {
    do {
        let book = try await attemptGeneration(
            concept: concept,
            pageCount: pageCount,
            onProgress: onProgress
        )
        return book
    } catch let error as LanguageModelSession.GenerationError {
        if case .guardrailViolation = error,
           attempt < GenerationConfig.guardrailRetryAttempts {
            // Retry with a fresh session
            continue
        }
        throw error
    }
}
```

## Content safety

All story concepts are sanitized before generation to ensure family-friendly content:

- Removes sensitive keywords
- Limits prompt length
- Ensures age-appropriate vocabulary
- Filters unsafe character combinations

## Character consistency

The system generates a `characterDescriptions` field that serves as a visual reference sheet:

```swift
var characterDescriptions: String
// Example: "Luna - small orange fox, green scarf, curious amber eyes"
```

This field is used later in the pipeline to ensure characters look consistent across all illustrations.

## Availability states

The story generator checks Foundation Model availability and provides helpful error messages:

| State | Description | User message |
|-------|-------------|-------------|
| Available | Model ready to use | Story generation enabled |
| Device not eligible | Non-Apple Silicon Mac | "This Mac doesn't support Apple Intelligence" |
| Not enabled | Apple Intelligence disabled | "Please enable Apple Intelligence in System Settings" |
| Model not ready | Still downloading | "The language model is still downloading" |

## Generation options

The system uses optimized generation parameters:

```swift
let textOptions = GenerationOptions(
    temperature: Double(GenerationConfig.defaultTemperature),
    maximumResponseTokens: GenerationConfig.foundationModelTokens(for: pageCount)
)
```

- **Temperature**: Controls creativity (0.0-1.0, lower is more focused)
- **Max tokens**: Dynamically calculated based on page count to ensure complete stories

## Story structure

Every generated storybook includes:

<AccordionGroup>
  <Accordion title="Title">
    A captivating title generated by the AI based on your story concept.
  </Accordion>
  
  <Accordion title="Author line">
    Defaults to "Written by StoryFox" unless you specify a custom author.
  </Accordion>
  
  <Accordion title="Story pages">
    Each page contains:
    - Page number (starting from 1)
    - 2-4 sentences of narrative text
    - An image prompt for illustration generation
  </Accordion>
  
  <Accordion title="Story moral">
    A one-sentence summary of the story's theme or lesson.
  </Accordion>
</AccordionGroup>
